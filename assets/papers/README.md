## Check before hosting the final PDF (subscription journals)

For these, the publisher usually holds copyright in the **published version**, so posting that exact
PDF on a personal site is typically **not** allowed. You can almost always host your **accepted
manuscript** (the post-review, pre-typeset version), sometimes after an embargo, and you can always
link the DOI (the site already does). Check each publisher's self-archiving policy:

- Nature 2025 (Metallohydrolases) and Nature Methods 2025 (RFdiffusion2): Springer Nature policy
- Adv. Healthcare Materials 2025: Wiley policy
- Springer book chapter 2023: Springer policy

Tip: SHERPA Romeo (v2.sherpa.ac.uk/romeo) summarizes each journal's self-archiving rules in one place.

If you want the accepted-manuscript versions on the site too, name them the same way (e.g.
`2025-metallohydrolases-nature-accepted.pdf`) and add a matching PDF button in `publications.html`.
